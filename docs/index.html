Features

Smart Editor

  • Multi-tab editing with session restore
  • Word prediction (bigram model)
  • Auto-indent & bracket closing
  • Line numbers & current line highlight
  • Customizable keyboard shortcuts
  • Subtle control characters (NEW!)

Wiki-Style Linking

  • Connect notes with [[wiki links]]
  • Automatic backlinks panel
  • File inclusion with [[!note]]
  • Ctrl+Click to follow links
  • Broken link detection

Task Management

  • Hierarchical task lists - [ ]
  • Auto-update parent tasks
  • Toggle with Ctrl+Space
  • Visual task states (pending/done/partial)
  • Nested subtasks support

Live Preview

  • Real-time markdown rendering
  • LaTeX formulas (KaTeX)
  • Mermaid diagrams
  • Code syntax highlighting
  • Custom CSS styling

Navigation

  • File tree view
  • Document outline panel
  • Quick open (Ctrl+P) with fuzzy search
  • Full-text search across files
  • Recent folders menu

Themes & Export

  • Light, dark, and system themes
  • Multiple editor color schemes
  • Export to PDF, HTML, DOCX
  • Preview themes (light/dark/sepia)
  • Customizable fonts

Getting Started

Windows (Portable Binary)

  1. Download TreeMk-0.2.0-Windows-x64-Portable.zip from releases
  2. Extract anywhere
  3. Run treemk.exe
  4. No installation required!

Build from Source

# Clone repository
git clone https://github.com/jailop/treemk.git
cd treemk

# Build
mkdir build && cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
cmake --build .

# Run
./treemk

Requirements

Qt 6.2+ CMake 3.16+ C++17 md4c Pandoc (optional)
